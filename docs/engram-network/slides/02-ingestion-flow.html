Entity Resolution
"home server" matches existing entity home-server-host → merge, don't duplicate. Strengthens edge to existing deployment-target engram.
Contradiction Handling
New fact "runs on Tailscale" contradicts old "Cloudflare tunnel active" → contradicts edge created. Consolidation daemon resolves later (newer wins by default).
Async, Zero Latency
Decomposition runs in background worker AFTER the conversation turn completes. User never waits. Raw input buffered in Redis, consumed by ingestion worker.
